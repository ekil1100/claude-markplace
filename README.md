# claude-marketplace

A Claude Code plugin marketplace.

## Install

```bash
claude marketplace add https://github.com/ekil1100/claude-markplace/raw/main/marketplace.json
```

Then install plugins:

```bash
claude plugin install planning-with-files@claude-marketplace
```

## Plugins

| Plugin | Description | Source |
|--------|-------------|--------|
| [planning-with-files](./planning-with-files/) | Manus-style persistent markdown files for planning and progress tracking | [OthmanAdi/planning-with-files](https://github.com/OthmanAdi/planning-with-files) |
